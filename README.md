# 🚀 Project Overview

**Pro Courier** is a modern courier and logistics management platform built to streamline parcel delivery services. The application enables customers to book deliveries, monitor shipment progress, and manage parcel history, while administrators oversee riders, customers, deliveries, and system activities through a centralized dashboard.

The project focuses on improving delivery efficiency, reducing manual operations, and providing a reliable experience for both customers and delivery personnel.

---

# 🎯 Goal

To develop a secure, scalable, and user-friendly courier management system that simplifies parcel booking, delivery tracking, rider management, and administrative operations through a centralized digital platform.

# ⚙️ Installation & Setup

## Clone Repository

```bash
git clone <https://github.com/kawsaralidev/pro-curier-client.git>
```

## Navigate to Project

```bash
cd pro-courier-client
```

## Install Packages

```bash
npm install
```

## Configure Environment Variables

Create a `.env` file in the project root.

```env
REACT_APP_API_URL=

REACT_APP_FIREBASE_API_KEY=
REACT_APP_FIREBASE_AUTH_DOMAIN=
REACT_APP_FIREBASE_PROJECT_ID=
REACT_APP_FIREBASE_STORAGE_BUCKET=
REACT_APP_FIREBASE_MESSAGING_SENDER_ID=
REACT_APP_FIREBASE_APP_ID=
```

## Run Development Server

```bash
npm start
```

## Build Project

```bash
npm run build
```
